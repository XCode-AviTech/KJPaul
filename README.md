# Kabonge's Portfolio Website

A modern and responsive personal portfolio website for **Kabonge**, an **Architect** and **Civil Engineer**.  
The site highlights his professional background, skills, services, and completed projects, providing a digital presence for clients and collaborators.

## 🌟 Features
- Clean and responsive design
- Projects showcase with image sliders
- Animated counters for years of experience and achievements
- Services section with icons
- About and contact sections
- Smooth scrolling and interactive effects

## 🛠️ Built With
- HTML5, CSS3, JavaScript
- [Bootstrap 5](https://getbootstrap.com/) for responsive layout
- [Swiper.js](https://swiperjs.com/) for image sliders
- Icons from [Bootstrap Icons](https://icons.getbootstrap.com/)

## 📌 Purpose
This portfolio serves as Kabonge’s professional online presence, allowing potential clients and partners to:
- View his architectural and civil engineering projects
- Understand his services and expertise
- Get in touch easily

## 🚀 Deployment
The site can be hosted on:
- GitHub Pages  
- Netlify  
- Vercel
